# README

Boilerplate per siti gestiti tramite Netlify CMS.

## Librerie esterne
### SASS
* Modular Scale
* Typi
* Susy

## Comandi
* `gulp dev`: esegue ambiente di sviluppo con browser-sync. Compila e controlla (watch) file `.pug`,`.scss` e `.js`.
* `gulp build --prod`: comando per la build.
